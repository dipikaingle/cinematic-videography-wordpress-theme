## 📌 Overview
Cinematic Videography is a custom WordPress theme designed for videographers, filmmakers, and creative professionals.

The theme focuses on delivering an immersive visual experience with modern layouts, smooth design, and strong storytelling elements.

Ideal for:
- Videographers
- Film studios
- Creative agencies
- Portfolio websites

---

## ✨ Features
- Fully custom WordPress theme (no page builder dependency)
- Modern and cinematic UI design
- Fully responsive layout (mobile, tablet, desktop)
- Video-focused sections and media integration
- Portfolio and project showcase
- Smooth user experience and navigation
- SEO-friendly and optimized structure
- Cross-browser compatibility

## ⚙️ Installation
1. Download or clone the cinematic-videography-pro
2. Upload the theme folder to:
   `/wp-content/themes/`
3. Go to WordPress Admin → Appearance → Themes
4. Activate the theme
